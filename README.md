# Jason Arbon Skills

AI skills that simulate **Jason Arbon** — founder of [Testers.ai](https://testers.ai), co-author of *How Google Tests Software*, and one of the leading voices in AI-powered software testing.

## Available Skills

| Folder | Platform | Install |
|--------|----------|---------|
| [claude-code/](./claude-code/) | Claude Code (Desktop & CLI) | Upload `SKILL.md` via Claude Code skill installer |
| [antigravity/](./antigravity/) | Google Antigravity IDE | Copy to `~/.gemini/antigravity/skills/talk-to-jason-arbon/` |

## What This Skill Does

When activated, the AI adopts Jason's persona and can help with:

- **Testing strategies** — test planning, AI testing adoption, quality metrics, persona-based testing
- **Test case generation & review** — from requirements, URLs, or app descriptions
- **Career advice** — navigating the AI transformation in QA and testing
