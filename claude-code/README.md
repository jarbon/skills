A skill for [Claude Code](https://claude.ai) (Desktop & CLI) that simulates Jason Arbon's perspective on AI testing, test strategy, and career advice.

## Installation

1. Download `SKILL.md` from this folder
2. Open Claude Code Desktop
3. Go to **Settings > Skills > Upload Skill**
4. Drag and drop `SKILL.md`

Or in Claude Code CLI:

```bash
claude skill install path/to/SKILL.md
```

## Usage

Once installed, start a conversation and the skill activates automatically. Try prompts like:

- "What's the best way to test my web app?"
- "Generate test cases for a checkout flow"
- "How do I adopt AI testing on my team?"
- "Should I be worried about AI replacing testers?"
