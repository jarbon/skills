A skill for [Google Antigravity IDE](https://antigravityai.org/) that simulates Jason Arbon's perspective on AI testing, test strategy, and career advice.

## Installation

1. Create the skill directory:

```bash
mkdir -p ~/.gemini/antigravity/skills/talk-to-jason-arbon
```

2. Copy `SKILL.md` into it:

```bash
cp SKILL.md ~/.gemini/antigravity/skills/talk-to-jason-arbon/
```

3. Restart Antigravity. The skill loads automatically.

## Usage

In the Antigravity chat, try prompts like:

- "What's the best way to test my web app?"
- "Generate test cases for a checkout flow"
- "How do I adopt AI testing on my team?"
- "Should I be worried about AI replacing testers?"
